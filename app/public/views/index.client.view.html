Fluff Link

This web app helps users rich share. What is rich sharing? Rich sharing is when your social media post include an image, description, title and other metadata. Think facebook previews.

Rich sharing is easy when you run your own site but some people who lease a website where they don't control the complete frontend, the metadata, or people who have blogs on single page applications with different routes have difficulty posting the content they want. This application helps you edit your post to be shared exactly the way you like and with the content you want.
    How it works
  • Fill in the description, author, and title you want to show on social media
  • Fill in the image url you want users to see, and the link url where it should go
  • An alert with your special link will show, be sure to copy that link and use it to share on social media
  • This will create the fluff link and redirect you to your site
  • Your link is now rich shared
